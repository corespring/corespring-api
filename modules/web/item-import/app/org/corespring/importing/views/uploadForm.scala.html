Item Import

Upload a zip containing IMS QTI with a qtimanifest.xml file. If this file is not included, the zip will be assumed to contain CoreSpring JSON.

  • Select this option to return a ZIP file whose contents can be uploaded using the corespring-batch-importer.

  • Select this option to upload the items into the database. This will return a CSV file containing the ids and URLs of all uploaded files.
